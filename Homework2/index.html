In the "Overview" chart to the left, I have created a scatterplot of the total points won for the winner and the loser for every single match in the dataset. When you hover over any point, you will see the winner name (total) vs. loser name (total) for that particular match. If you click on the point, the "Detail" chart to the right will display the statistics of each player for that match. These statistics can be changed by the radio buttons, and the default is set to the First Serve percentage. In order to change the detail view, you must first change the radio button selection, then click on the data point you are interested in.

Note that the default "Detail" chart is the first record in the dataset, and that null values are simply displayed as 0.
